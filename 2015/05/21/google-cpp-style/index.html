google C++ 代码风格

  • tech

最近在写 C++ 程序，总感觉以前的 C 风格好像不太够用，突然想到了以前看的 Google C++ 风格，整理一下，以后就按 Google C++ 风格来!

一个漂亮的代码风格，在看的时候就会很舒服，而且团队代码风格一致，更是一种享受！

作为一个理智的 Google 脑残粉，选代码风格，肯定直接上 Google 了，更何况在 Google 要管理这么多代码，总结出来的代码风格肯定是毋庸置疑的。
