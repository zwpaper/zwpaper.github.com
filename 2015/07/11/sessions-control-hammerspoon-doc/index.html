Sessions control for hammerspoon

  • Software
  • tech

Sessions control 是一套基于 hammerspoon(Mac) 的脚本程序。可以通过快捷键把 Mac 中的各个窗口归类，并快速的在各个分类中切换，包括全屏的窗口也会自己切换。

**注意：**本程序还处于开发状态，不会关闭或丢失窗口，但并**不对**稳定性提供保证，**请勿**用于正式工作中！

Hammerspoon

sessions control

简介

Sessions control 主要有以下三个功能：

  • 显示当前情景(sessions)
  • 在不同情景中切换
  • 为当前情景增删窗口(windows)

当前实现主要通过把窗口最小化完成情景切换，后期版本会考虑把窗口隐藏，不占用任务栏空间。

目前主要优势在于同时可以处理全屏窗口，当切换情景时，全屏的窗口也会同时最小化，前台情景也会恢复应有的全屏状态。

各个情景都会定时保存到磁盘中，以保证各个情景不会因为 hammerspoon 或电脑的开关而丢失。

对各个功能的快捷键提供定制，在脚本中可以很简单地修改快捷键。

功能说明

在本说明中使用默认的快捷键进行说明，操作主要是先按住功能键，然后点击相应快捷键，默认键位如下：

  • 功能键：
    ctrl + alt + cmd

  • 显示当前情景：P

  • 前一情景：[
  • 下一情景：]
  • 加入当前情景：L
  • 从当前情景删除：；

  • 数字键快速切换：1 ~ 情景数

    其中，数字键快速切换 会按照情景数自动生成，通过按住功能键，加上相应的数字可以快速在情景间切换。

    默认键位都位于键盘右侧，通过功能分区。

显示当前情景

按住 ctrl + alt + cmd 后，点击 P

通过系统通知展示当前各个情景，并自动编号，便于后面的快速切换。

当然，同时进行太多任务会很乱，所以不应该同时有太多情景，同时，系统通知中不可以放入过长的内容，所以刚刚好 (;¬_¬)

前后情景

  • 前一情景：按住 ctrl + alt + cmd 后，点击 [
  • 下一情景：按住 ctrl + alt + cmd 后，点击 ]

这两个功能主要用于切换上一个或下一个情景，

程序实现

Sessions control 是基于 Hammerspoon 的脚本程序，使用 lua 编写，主要是

  • sessions_control.lua
  • sessions_head.lua

两个文件实现。
